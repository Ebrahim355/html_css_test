خصائص النصوص

تحديد عائلة الخط المستخدم للنص.

هذا النص باستخدام Arial

هذا النص باستخدام Times New Roman

هذا النص باستخدام Courier New

/* خط بدون serif */

    font-family: 'Arial', sans-serif;
}

/* خط مع serif */

    font-family: 'Times New Roman', serif;
}

/* خط monospace */

    font-family: 'Courier New', monospace;
}

div {
    position: fixed;

white-space

التحكم في كيفية معالجة المساحات البيضاء في النص.

normal (افتراضي)

هذا النص يحتوي على مسافات متعددة وسيتم طيه تلقائياً

nowrap

هذا النص يحتوي على مسافات متعددة ولن ينكسر إلى سطر جديد

pre

هذا النص يحافظ على المسافات كما هو مكتوب في الكود

pre-wrap

هذا النص يحافظ على المسافات لكنه ينكسر عند الحاجة

pre-line

هذا النص يدمج المسافات المتعددة لكنه يحافظ على فواصل الأسطر 
div {
    white-space: normal; /* افتراضي - يطوي المسافات الزائدة */
}

div {
    white-space: nowrap; /* لا يسمح بفصل النص إلى أسطر جديدة */
}

div {
    white-space: pre; /* يحافظ على المسافات كما في الكود */
}

div {
    white-space: pre-wrap; /* يحافظ على المسافات مع كسر السطر عند الحاجة */
}

div {
    white-space: pre-line; /* يدمج المسافات الزائدة مع الحفاظ على فواصل الأسطر */
}

word-break

هذهكلمةطويلةجداوسيتمكسرهافيأيمكان

التخطيطات الحديثة

multi-column layout

تقسيم المحتوى إلى عدة أعمدة مثل الجرائد.

هذا نص طويل سيتم تقسيمه إلى ثلاثة أعمدة. خاصية multi-column layout تتيح لك تقسيم المحتوى إلى عدة أعمدة مثل الجرائد والمجلات. يمكن التحكم بعدد الأعمدة والعرض والفراغ بينها وإضافة خط فاصل بين الأعمدة.

هذه الخاصية مفيدة لعرض النصوص الطويلة بطريقة أكثر قابلية للقراءة وتوفير المساحة. يمكن أيضًا التحكم في كيفية تدفق المحتوى بين الأعمدة وإضافة عناوين تمتد عبر جميع الأعمدة.

.container {
    column-count: 3; /* عدد الأعمدة */
    column-gap: 20px; /* المسافة بين الأعمدة */
    column-rule: 1px solid #ddd; /* خط فاصل بين الأعمدة */
    column-width: 200px; /* يمكن استخدامها بدلاً من column-count */
}

/* عنصر يمتد عبر جميع الأعمدة */
.full-width {
    column-span: all;
}
